# camera-tag-algo
记录研究日志如下
高点球机标签移动算法

带PTZ的相机，给某物体打上标签，移动后标签跟随物体，推导成像方程效果如下
1. 缩放无偏移
2.边缘偏差很小
3.支持各种相机
4.几乎不消耗资源 

待解决：如何优化至零偏差，目前还在优化中
